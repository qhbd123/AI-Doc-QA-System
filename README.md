# AI 智能文档问答系统 

基于 RAG（检索增强生成）的企业级文档问答系统，支持多格式文档上传、BM25 检索、多模型自动降级、Redis 缓存、多轮对话等特性。

## 功能特点

- 📄 **多格式文档解析**：支持 PDF、DOCX、TXT、Markdown、HTML，自动清洗并语义化分块。
- 🔍 **高效检索**：基于 BM25 算法，支持文档选择过滤，提供相关性分数。
- 🤖 **多模型支持**：集成豆包（Doubao）、DeepSeek、智谱清言，可运行时动态切换，主模型失败时自动降级。
- ⚡ **智能缓存**：Redis 缓存问答结果，缓存 key 包含模型、问题、文档签名，避免串用，提供命中率统计。
- 💬 **多轮对话**：基于 Redis 存储会话历史，支持上下文记忆，可清空对话。
- 🎨 **现代化前端**：三套主题（浅色/深色/蓝灰），拖拽上传，文档勾选，停止生成，缓存徽章。

## 技术栈

- **后端**：FastAPI（异步）+ Uvicorn
- **检索**：BM25（rank_bm25）+ jieba 分词
- **缓存**：Redis（存储问答缓存 + 会话历史）
- **大模型 API**：豆包、DeepSeek、智谱清言（OpenAI 兼容格式）
- **前端**：原生 HTML/CSS/JS，CSS 变量主题

## 环境要求

- Python 3.11+
- Redis 服务（本地或远程）
